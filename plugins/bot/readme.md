### Introduction

# Bot

> 管理地毯模组假人

## 使用方法

`!!bot` 显示假人列表

`!!bot help` 显示帮助

`!!bot spawn <name>` 生成假人

`!!bot kill <name>` 移除假人

`!!bot add <name> <dim> <pos> <facing>` 添加假人到假人列表

`!!bot remove <name>` 从假人列表移除假人
